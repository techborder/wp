== Description ==
Wedding is an elegant, responsive WordPress theme designed for wedding websites. The theme includes multiple pages, homepage slider and gallery support.  It uses special pages to deliver the story of the bride and the bridegroom, as well as wedding details and to create online invitations. The theme can also be used for the wedding photographers and any other event organization websites with its capacity of editable color schemes, layout and styles. The marketing component can be ensured with the help of testimonials and advertisement features.


== Installation ==
You can install the theme through the WordPress installer under Themes-Install themes by searching for it.
Alternatively you can download the file from here, unzip it and move the unzipped contents to the wp-content/themes folder
of your WordPress installation. You will then be able to activate the theme.

You can using theme settings page in admin side to setting up our theme.
