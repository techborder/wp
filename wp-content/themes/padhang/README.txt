== Description ==
A minimalist-content-focused theme. Padhang comes with responsive layout, 2 widget areas under the main content, social menu, big background image, custom logo and custom menu. Padhang is a Javanesse word for bright. It featured bright white color with dark background. Easily customized from the theme customizer. Stand out your content.

== Features ==
* Minimalist-content-focused design
* Responsive layout
* Custom header
* Custom background
* Custom color via theme customizer
* Custom menu
* Social menu
* 2 widget areas
* Translation-ready
